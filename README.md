# Prototype-UI-Design
 Project Title (Prototype UI Design)
Project Overview

This project is a UI/UX prototype based on UML Use Case Diagram (DCU).
The goal is to transform functional analysis into interactive user interface screens.

 Tools Used
Figma (Prototype Design)
UML (Use Case Diagram Analysis)
Features
User interface screens based on use cases
Interactive navigation between screens
Role-based system design (if applicable)
Clickable prototype simulating real application flow
 Prototype Link
https://www.figma.com/design/SHqcDkPkPmtCXodd08ni80/GL-PROTOTYPE?node-id=0-1&t=WpJRT6dyPbgnyJQ4-1
 Learning Outcomes
Understanding of functional analysis (UML DCU)
Conversion of requirements into UI screens
Navigation flow design
Prototyping using Figma

 Note

This is a UI/UX academic prototype project, not a fully coded application.
